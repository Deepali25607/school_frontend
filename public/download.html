How to install

  1. Tap Download for Android above.
  2. Open the downloaded Lumina.apk file.
  3. If prompted, allow “Install unknown apps” for your browser, then tap Install.
  4. Open Lumina and sign in.

This app is distributed directly (not via the Play Store), so Android's "unknown apps" prompt is normal.
